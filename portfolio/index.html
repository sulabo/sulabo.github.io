일하는 순서는 늘 같습니다. 직접 다 해보고, 기준을 세우고, 문서로 남기고, 위임하고, 마지막에 자동화합니다. 요즘은 사람 손이 많이 가던 운영을 AI로 줄이는 데 집중하고 있습니다. 2026년 Snowflake AI & Data 해커톤 비즈니스 트랙에서 1위를 한 것도 같은 흐름입니다.The order is always the same: do it all myself, set the criteria, write it down, delegate, and automate last.
